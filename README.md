Clonar el repositorio:
git clone https://github.com/TU-USUARIO/proyecto-crean.git

Entrar al proyecto:
cd proyecto-crean

Instalar dependencias:
npm install

Crear archivo .env basado en .env.example

Ejecutar el servidor:
npm run dev

Ejemplo (.env.example):

PORT=3000
DB_HOST=
DB_USER=
DB_PASSWORD=
DB_NAME=
JWT_SECRET=
